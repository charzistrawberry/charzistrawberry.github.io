EnsembleDroid

A malware detection approach that is able to determine whether an Android application is considered malicion or benign. Two classifiers were utilized based on Ensemble Learning. The structure of the classifier starts with Android apps in the form of APK files which is decompiled into several readable files. The manifest file of each app is extracted using a python script that creates an XML file of binary vectors where each vector represents the permissions that app utilizes. That XML file is split 80/20 where 80% is used for training and 20% is used for testing. A 10 fold cross validation procedure was used for analysis of the classifiers' results. Both homogenous and heterogeneous stacking classifiers produced results of up to 91% using machine learning algorithims such as Support Vector Machines, K-Neighest Neighbor, Linear Regression, and Logistic Regression.


Key Words: Machine Learning, Malware Detection, Android System, Ensemble Learning

Long Island Railroad Simulation

Simulation that replicates the system of the Long Island railroad. Users can see the train travel times and arrival times at each station. Users can also keep track of the number of people in each cart as well as on the platforms waiting at each station. Capacity of space is considered in the cart and will pose an alert if number of passengers exceen the limit. Program utilizes foundations of data structures and algorithms as well as object-oriented programming.


Key Words: Java, Data Structure & Algorithims, Object-Oriented Programming, Java
